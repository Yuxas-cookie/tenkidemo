# 塗装工程AIスケジューラー v3 仕様書

- **ステータス**: 確定
- **作成日**: 2026-04-08
- **目的**: v2にスケジュール提案・カレンダー・代替プラン機能を追加

---

## 概要

### なぜ必要か
v2はダッシュボード型で見やすくなったが、AIが「1つの最適化結果」を返すだけだった。
営業デモとして「複数プランを提案→選択→カレンダーに登録→当日の急な雨にも対応」という実用的なフローを見せたい。

### 新機能
1. **複数スケジュール提案**: AIが3つのプラン（最速・バランス・安全）を提案
2. **カレンダーページ**: 登録済みスケジュールをカレンダー形式で一覧表示
3. **スケジュール登録**: 提案プランを選んで「採用」→カレンダーに反映
4. **代替プランストック**: 不採用プランも保存、いつでも切り替え可能
5. **当日雨シミュレーション**: 特定日を「雨に変更」して影響をリアルタイム確認

---

## 確定した仕様

### 複数プラン提案
- AIが3つのプラン（A/B/C）を生成
  - **プランA「最速」**: 工期最短優先、リスク高め
  - **プランB「バランス」**: 工期とリスクのバランス
  - **プランC「安全」**: 天候リスク最小化、工期長め
- 各プランにスコア表示: 工期日数・リスクレベル・追加コスト
- ユーザーがプランを選んで「このプランを採用」ボタン

### カレンダーページ `/calendar`
- 月表示のカレンダーグリッド
- 登録済みの全現場のスケジュールを色分けで表示
- 各日にどの現場のどの工程が予定されているかを表示
- 天気予報のオーバーレイ（施工不可日をハイライト）

### スケジュール登録（状態管理）
- 採用プランをグローバルstateに保存（Context Provider）
- 現場ごとに「採用プラン」と「代替プラン（ストック）」を保持
- カレンダーには採用プランのみ表示

### 代替プランストック
- 不採用のプランも `alternativePlans` として保存
- 現場詳細ページで「代替プラン一覧」として表示
- ワンクリックで代替プランに切り替え可能

### 当日雨シミュレーション
- カレンダーまたは天気予報ページで特定日をクリック→「雨に変更」
- 変更後のスケジュール影響を即座に表示
- 「このまま雨の場合の代替スケジュール」をAIが即座に生成

## ページ構成（v3）

1. **ダッシュボード** `/` — 統計・アラート・最近の現場
2. **現場一覧** `/sites` — 全現場カードリスト
3. **現場詳細** `/sites/[siteId]` — 工程表・採用/代替プラン
4. **天気予報** `/weather` — 16日間予報 + 雨シミュレーション
5. **AIシミュレーション** `/simulation` — 複数プラン提案 + 採用
6. **カレンダー** `/calendar` — 登録済みスケジュール一覧

### サイドメニュー追加項目
- Calendar アイコン: カレンダー `/calendar`
