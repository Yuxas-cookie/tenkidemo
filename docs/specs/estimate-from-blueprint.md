# 設計図から見積作成機能 仕様書

- **ステータス**: 確定
- **作成日**: 2026-04-08
- **目的**: 建物の図面（PDF/画像）をアップロードし、AIが分析して塗装面積・予算を自動算出

---

## 概要

### フロー
```
設計図アップロード → Claude Vision APIで分析 → 面積・建物情報を自動抽出
→ 塗料グレード選択 → 詳細見積内訳を自動算出 → 見積結果表示
```

### 画面構成
- **新規ページ**: `/estimate` — 見積作成
- **サイドメニュー追加**: 「見積作成」（FileText アイコン）
- **ダッシュボードStep1**: 「設計図から入力」ボタン追加

## 見積算出ロジック（外壁塗装の一般的な単価）

### 工程別単価（m²あたり・円）

| 工程 | シリコン | フッ素 | 無機 |
|------|---------|--------|------|
| 足場設置・撤去 | 800 | 800 | 800 |
| 飛散防止シート | 200 | 200 | 200 |
| 高圧洗浄 | 250 | 250 | 250 |
| 下地処理（ケレン） | 600 | 600 | 600 |
| シーリング工事 | 900 | 900 | 900 |
| 養生 | 350 | 350 | 350 |
| 下塗り | 800 | 800 | 800 |
| 中塗り+上塗り（塗料代含む） | 2,800 | 4,200 | 5,500 |
| 付帯部塗装 | 1,000 | 1,200 | 1,400 |
| 諸経費（材料運搬・廃材処理等） | 500 | 500 | 500 |

### 塗料グレード
1. **シリコン塗料** — 耐用年数10〜13年、コスパ最良
2. **フッ素塗料** — 耐用年数15〜18年、高耐久
3. **無機塗料** — 耐用年数20〜25年、最高級

### 建物種別の係数
- 戸建て: ×1.0
- マンション: ×0.85（スケールメリット）
- 公共施設: ×1.1（品質基準が高い）

### 算出式
```
各工程費 = 塗装面積 × 単価 × 建物種別係数
小計 = 全工程費の合計
消費税 = 小計 × 10%
総額 = 小計 + 消費税
```

## AIによる設計図分析

### Claude Vision APIに送信する内容
- アップロードされた画像/PDF
- プロンプト: 「この建物の図面から以下を推定してください: 建物種別、延べ床面積、外壁面積（塗装面積）、階数、構造（木造/RC/鉄骨）」

### AIが返すJSON
```json
{
  "buildingType": "house" | "apartment" | "public",
  "floors": 2,
  "structure": "木造" | "RC造" | "鉄骨造",
  "totalFloorArea": 120,
  "estimatedPaintArea": 150,
  "description": "2階建て木造住宅。延べ床面積約120m²、外壁塗装面積は約150m²と推定",
  "notes": ["バルコニーの塗装面積を含む", "屋根は別途見積が必要"]
}
```

### APIキー未設定時
モックデータを返す（デモ用）

## UI設計

### 見積作成ページ `/estimate`

#### Step 1: 設計図アップロード
- ドラッグ&ドロップエリア（大きく）
- または「ファイルを選択」ボタン
- 対応形式: PNG, JPG, PDF
- アップロード後: サムネイルプレビュー表示

#### Step 2: AI分析結果
- 建物情報カード（種別・階数・構造・面積）
- AI推定の説明文
- 面積の手動調整スライダー

#### Step 3: 塗料グレード選択
- 3つのカード（シリコン/フッ素/無機）
- 各カード: 耐用年数・m²単価・特徴

#### Step 4: 見積結果
- 工程別内訳テーブル（大きく見やすく）
- 小計・消費税・総額
- 「スケジュールシミュレーターに連携」ボタン（ダッシュボードへ遷移）
