# 塗装工程AIスケジューラー v2 仕様書

- **ステータス**: 確定
- **作成日**: 2026-04-08
- **目的**: v1のUI/UXを全面刷新。サイドメニューベースのダッシュボードアプリに再設計

---

## 概要

### なぜ作り直すか
v1は文字が小さく、フォントが見にくく、ページ構造がフラットで営業デモとしてわかりにくかった。サイドメニュー型のダッシュボードアプリとして再設計し、直感的に操作できるUIにする。

### 設計方針
- **サイドメニューナビゲーション**: 常に表示される左サイドバーで各機能へ即アクセス
- **大きく読みやすいフォント**: Noto Sans JP（日本語）+ Inter（英数字）
- **ページ分割**: 機能ごとに独立したページ
- **余白たっぷり**: デモで見せやすい余裕のあるレイアウト
- **カラーテーマ**: 白背景 + ブルー系アクセント + パープルのAI演出

---

## 技術スタック（変更なし）
- Next.js 16 + TypeScript + Tailwind CSS 4
- shadcn/ui + Framer Motion
- Claude API (Anthropic SDK)
- Open-Meteo API（天気）

## ページ構成（サイドメニュー対応）

### サイドメニュー項目
1. **ダッシュボード** `/` — 全体概要
2. **現場一覧** `/sites` — 全現場のリスト
3. **天気予報** `/weather` — 天気予報の詳細表示
4. **AIシミュレーション** `/simulation` — AI最適化の実行

### 個別ページ（サイドメニューから遷移）
5. **現場詳細** `/sites/[siteId]` — 工程表・ガントチャート

---

## 画面詳細

### 1. サイドメニュー（全ページ共通）
- 左固定幅（280px）
- ロゴ + アプリ名
- ナビゲーションリンク（アイコン + テキスト）
- 下部: 天気モード切替（リアル/デモ）
- アクティブページのハイライト

### 2. ダッシュボード `/`
- 統計カード（現場数・進行中・天気アラート・AI最適化回数）
- 直近の天気アラート（簡易版）
- 最近の現場ステータス（進行中の現場を3つ表示）

### 3. 現場一覧 `/sites`
- テーブル or カードリスト形式
- 現場名・住所・種別・面積・ステータス・進捗率
- クリックで現場詳細へ

### 4. 天気予報 `/weather`
- 16日間予報の大きなカレンダー表示
- 各日の詳細（天気・気温・湿度・風速・施工可否）
- 施工不可日のハイライト

### 5. 現場詳細 `/sites/[siteId]`
- 物件情報
- ガントチャート（工程スケジュール）
- 工程リスト
- 天候影響サマリー

### 6. AIシミュレーション `/simulation`
- 現場選択ドロップダウン
- 事前シミュレーション / 緊急リスケのタブ
- AI実行ボタン
- Thinking演出 → 推論表示 → Before/After比較 → 提案カード
